## Node.js 설치

---

https://nodejs.org/en/

위 링크에서 node.js 16.xx버전으로 설치.

## 프로젝트 설치

---

node: 16.13.0 LTS 기준

git clone후 폴더 내에서 npm install 실행

```
git clone https://github.com/AI-Funding/AI_Funding_FE.git
```

```
cd AI_Funding_FE
```

```
npm install
```

npm install시 나타나는 vulnerabilities 경고는 무시해도 됩니다.  
(npm 관련 문제라 프로젝트에 영향 x)
